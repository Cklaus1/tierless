## Grader notes
The headline POC finding: **skills-arm B scored LOWER than bare-arm A (0.80 vs 1.00)**
on this single run — because it missed the org_report IDOR that A caught, and did so
confidently ("the only one doing it correctly"). Two lessons:
1. **Single-run variance is real and can invert the expected ordering.** This is the
   empirical case for the rubric's ">=3 runs per arm, report the spread" rule. One run
   proves nothing; it can even mislead.
2. The security-review skill's report *format* clearly shaped B (structured findings,
   exploit paths, verdict) — the discipline showed up in the shape. But format
   adherence spent attention that A spent on coverage. Possible real effect worth
   testing at N: does the template help structure but hurt breadth on easy tasks?
Confirms Task 05 is at/near ceiling for a small model and doesn't discriminate cleanly.
